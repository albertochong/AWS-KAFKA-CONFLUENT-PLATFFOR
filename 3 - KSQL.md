* Some ksql commands
```bash
1 - list topics;   ------------------------------ list all topics
2 - print 'topic_name' -------------------------- print only all newest messages
    print 'topic_name' from beginning ----------- print all messages
    print 'topic_name' from beginning limit n --- print top n messages
```
